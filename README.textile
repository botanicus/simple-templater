h1. About

"SimpleTemplater":http://github.com/botanicus/simple-templater is simple solution for creating generators

h1. Usage

h2. API

<pre>
require "simple-templater"

SimpleTemplater::Project.new do
	self.name = "rango"
	self.logger = Rango.logger
end

SimpleTemplater::Generator.new("project")
</pre>

h1. Custom Generators

Each user can create its own generators and put them to @~/.#{project_name}/stubs/#{generator_name}@. For example project generators for Rango should be placed to ~/.rango/stubs/project

h2. Plugins

You often may want to

In Rango:
<pre>
SimpleTemplater::Discovery.discover!(:rango)
</pre>

<pre>
SimpleTemplater.scope(:rango) do
	register File.dirname(__FILE__)
end
</pre>
